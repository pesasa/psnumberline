Psnumberline examples

Here are couple of examples, how to use psnumberline.js.

<div class="psnl1"> </div> <script type="text/javascript"> jQuery('.psnl1').psnumberline({min: -20, max: 20, linewidth: 1.5}); jQuery('.psnl1').psnumberline('addpoint', {name: 'Point', shape: '[]'}); jQuery('.psnl1').psnumberline('addpoint', {name: 'A', value: -4.2, color: 'blue', shape: '+'}); jQuery('.psnl1').psnumberline('addpoint', { name: 'B', value: 6.9, color: 'gold', draggable: true, epsilon: 0.5 }); </script>
